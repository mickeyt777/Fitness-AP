FITNESS GLP · DESIGN KICKOFF · v1 (dark-first, light pass)

Fitness GLP — Reference Mockups

First-pass visual direction plus the four key screens (Today, Progress, Paywall, Onboarding). Built as static previews to react to — not production SwiftUI. Includes the new steps/cardio surface, a rest-day state, the activity sparklines, the trial framing, and the Watch + HealthKit onboarding steps. Light-mode variants come after we settle direction.

Visual identity — proposal

Palette

GLP Green#34D399
Mint tint#8FF0CE
Energy#FBBF6B
Cardio#FF8B6B
Data#A78BFA
Card#161F23
Base#0B1012

Green carries the muscle-retention / growth idea without going neon. Amber + coral are reserved strictly for activity and cardio so motion reads instantly. Violet tags measurement data. Supportive, clinical-adjacent — never hospital-sterile.

Type scale — SF Pro

Large Title34 / Bold
Title 222 / Bold
Headline17 / Semibold
Body17 / Regular
Subhead15 / Regular
Footnote / Caption13 · 11

SF Pro Display for titles, SF Pro Text for body — system default keeps it native to iOS 26 and free with Dynamic Type.

Iconography & app icon

Rounded SF Symbols at regular weight; duotone fills for rings. App-icon concept: an open arc (incomplete ring = "still building") cradling a plus — supportive progress, gentle clinical cue. Gradient green on near-black.
